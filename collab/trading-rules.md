## 九、复盘执行规则

### 复盘时间承诺

| 周期 | 时间要求 | 时长 |
|------|----------|------|
| 每日收盘后 | 收盘后15分钟内 | 15分钟（快速版）/ 30分钟（完整版） |
| 每周日 | 周度Lint | 30分钟 |
| 每月末 | 月度大Lint | 1小时 |

### 复盘核心原则

> ⚠️ **不以盈亏论对错，以是否符合系统论对错**

- 系统内亏损也是对的
- 系统外盈利也是错的

### 系统执行检查（每日必做）

| 检查项 | 标准 |
|--------|------|
| 符合选股标准吗？ | 标的在优先级列表中 |
| 符合买点条件吗？ | 符合定义的买点类型 |
| 仓位合适吗？ | 不超过单票/单赛道限制 |
| 逻辑清楚吗？ | 买入理由明确 |
| 能接受亏损吗？ | 已设止损，亏损可控 |

### 对错分析框架

**做对的交易：**
- 做对的地方
- 原因分析
- 如何复用

**做错的交易：**
- 做错的地方
- 原因分析
- 改进措施（必须具体可执行）

## 十、系统进化检验

### 三个月后应能清晰回答

**我的核心盈利模式是什么？**
- [ ] 容量票趋势交易
- [ ] 极值策略反弹
- [ ] 情绪周期套利
- [ ] 指数共振买点

**我最常犯的3个错误是什么？如何防范？**
- 情绪化交易 → 制定计划，盘中不临时起意
- 止损执行不严 → 使用条件单
- 横跳问题 → 专注一个方向

**我在什么市场环境下最应该空仓？**
- [ ] 外部冲击期
- [ ] 情绪退潮期
- [ ] 无主线机会期
- [ ] 连续亏损后调整状态
